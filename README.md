# Next.js App Router Learning Project

Bu proje, **Next.js App Router**, **React** ve **TypeScript** kullanılarak geliştirilen bir öğrenme projesidir. Proje kapsamında sayfa yönlendirme, bileşen yapısı, veri çekme ve CSS Modules gibi temel Next.js özellikleri uygulanmıştır.

## 🚀 Kullanılan Teknolojiler

- Next.js (App Router)
- React
- TypeScript
- CSS Modules
- Fetch API

## 📂 Proje Yapısı

```
app/
├── components/
│   ├── AddToCart.tsx
│   ├── ProductCard.tsx
│   └── ProductCard.module.css
├── users/
│   ├── page.tsx
│   └── new/
│       └── page.tsx
└── page.tsx
```

## ✨ Özellikler

- App Router ile dosya tabanlı yönlendirme
- Nested Routing (`/users/new`)
- Yeniden kullanılabilir React bileşenleri
- Client Component (`'use client'`)
- Server Component ile veri çekme
- Harici API'den kullanıcı listesini alma
- TypeScript interface kullanımı
- CSS Modules ile bileşen bazlı stil yönetimi

## 📚 Öğrenilen Konular

- Next.js App Router
- File-based Routing
- Client & Server Components
- Component Composition
- Fetch API ve Async/Await
- TypeScript Interfaces
- Dinamik listeleme (`map`)
- CSS Modules

## ▶️ Kurulum

```bash
git clone <repo-url>
cd <project-folder>
npm install
npm run dev
```

Uygulama varsayılan olarak **http://localhost:3000** adresinde çalışacaktır.
